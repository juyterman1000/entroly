Under the Hood

How Entroly works

A 5-stage pipeline that runs in under 10 milliseconds.

Ingest

Indexes your codebase, builds a dependency graph, fingerprints every code fragment for instant dedup. 847 files in 1.8 seconds.

Score

Ranks every fragment by information density. High-value code scores high. Boilerplate, duplicates, and dead code score low.

Select

Picks the mathematically optimal subset that fits your token budget. Not top-K. Not random. Provably optimal via knapsack solver with diversity constraints.

Deliver

Three resolution levels: critical files in full, supporting files as function signatures, peripheral files as one-line references. 100% coverage.

Learn

Tracks which context produced good AI responses. Reinforcement learning adjusts selection weights over time. Gets smarter the more you use it.
