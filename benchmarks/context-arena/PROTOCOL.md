# Context Arena — Protocol v1 (frozen)

Derived from the preregistered protocol in
`benchmarks/AGENTIC_TASKS_PREREGISTRATION.md` (entroly repository, locked
2026-07-20). Amendments require a declared v2 — never silent edits.

## 1. Task construction

- Source: public repositories with test suites.
- Candidate: a non-merge commit that modifies ≥1 source file and ≥1 test
  file, with ≤20 files changed (bulk refactors excluded).
- Validation (fail-to-pass), in an isolated worktree at the fix commit:
  1. the touched tests **pass** at the commit;
  2. after reverting **only the source files** to the parent (tests kept),
     the same tests **fail**;
  3. commits whose source files have no parent version are rejected.
- The failing test command is the task oracle. No LLM judging anywhere.

## 2. Arms

Same task, same model, same decoding parameters, same wall-clock/token caps:

| Arm | Description |
|---|---|
| NO-CONTEXT | baseline: task description only |
| ADAPTER-X | the context layer under test prepares the context |

Multiple adapters may run against the same task set; each is paired with the
baseline and with each other.

## 3. Metrics & statistics

- Task success (oracle passes), per arm.
- Input/output tokens and estimated cost per task.
- Paired success differences: **exact McNemar**.
- Token savings: paired bootstrap 95% CI.
- Dev/holdout split **by repository** — no repo contributes to both.

## 4. Power & sample size

With ~10% discordant pairs, n = 300 detects ≈6pp at 80% power; target
n ≥ 400 mined tasks for margins near 3pp. If mining yields fewer, widen the
declared margin **before** the run and record it as a protocol amendment.

## 5. Contamination control

- Record the fix-commit date for every task.
- Report results split by pre/post the evaluated model's training cutoff.
- Prefer post-cutoff commits when curating the headline set.

## 6. Reporting rules

- Preregistered thresholds only; commit them before the first run.
- All artifacts committed, including failed runs, with payload hashes.
- A claim that cannot be regenerated by one documented command is invalid.
