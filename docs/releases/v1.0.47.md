# Entroly 1.0.47

Entroly 1.0.47 restores the established SimHash/LSH token geometry in
the Rust deduplication engine. The MD5-derived stable 64-bit geometry
keeps persisted indexes compatible and prevents matching fragments
from becoming unreachable after a hash-family change.

This release synchronizes the Python, native Rust, QCCR, WASM, npm,
MCP, OpenClaw, plugin, Docker, and registry version surfaces.
