# Entroly 1.0.49

Entroly 1.0.49 ships replayable Context Commits and the stable
SimHash/LSH geometry compatibility fix on a fully synchronized release.

It also repairs the release-wide Python prerelease assertion and the
commit identity policy for GitHub-reviewed merges, restoring green CI
across Python 3.10 through 3.14 and the pure-Python fallback.

All active Python, Rust, QCCR, WASM, npm, MCP, OpenClaw, plugin,
runtime, lockfile, Homebrew URL, and release-test version surfaces are
generated together by the canonical bump script.
