# Entroly 1.0.56

Entroly 1.0.56 strengthens the product as a context control plane around
Claude Code, Codex, and OpenClaw. The release adds:

- trust-labelled GPT-5.6 and Nemotron metadata, plus opt-in OpenRouter and
  local model discovery that preserves unknown fields instead of guessing;
- scoped, project-bound, revocable, and expiring MCP attachment grants;
- synchronous restart replay for persisted Slack, Discord, and Telegram
  gateway deliveries;
- a searchable, read-only context-session dashboard with receipt integrity,
  omitted-evidence, model, token, and cost visibility.

Reliability behavior is explicit: partial client installation revokes access
and attempts rollback, attached MCP tools reauthorize every call, queued events
replay before an online announcement, corrupt artifacts remain read-only and
visible, and incomplete price or usage data is reported as unknown.

Release metadata is synchronized across Python, Rust, npm, MCP, OpenClaw,
WASM, plugin, Docker, Homebrew, and GitHub release surfaces.
