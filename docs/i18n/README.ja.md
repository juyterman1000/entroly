# Entroly

Entroly は AI エージェント向けのローカルで監査可能なコンテキスト層です。明示的なトークン予算の範囲でリポジトリの根拠を選択し、採用・省略した情報を示すレシートを生成します。

```bash
pip install entroly
cd /path/to/repository
entroly verify-claims
```

`verify-claims` は限定的なローカルのインストール／選択スモークテストです。コスト削減、回答品質、請求結果を保証するものではなく、結果はワークロードに依存します。
