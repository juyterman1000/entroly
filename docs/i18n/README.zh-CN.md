# Entroly

Entroly 是面向 AI 智能体的本地、可审计上下文层。它在明确的令牌预算内选择代码库证据，并生成收据，说明包含和省略了哪些内容。

```bash
pip install entroly
cd /path/to/repository
entroly verify-claims
```

`verify-claims` 只是有限的本地安装与选择冒烟测试，不保证节省成本、回答质量或账单结果。实际结果取决于具体工作负载。
