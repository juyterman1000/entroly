## Lo que realmente obtienes

| | Sin Entroly | **Con Entroly** |
|---|---|---|
| Archivos que ve la IA | 5–10 | **Todo tu repositorio** |
| Tokens por petición | ~186,000 | **9,300 – 55,000** |
| Costo por 1K peticiones| ~$560 | **$28 – $168** |
| Contexto efectivo | 200K | **~2M (compresión de resolución variable)** |
| Costo de aprendizaje | Crece (tokens) | **$0 — comprobablemente token-negativo** |
| Configuración | Horas de hacks | **30 segundos** |

Los archivos críticos se envían completos. Los de soporte como firmas (signatures). El resto como referencias. Tu IA obtiene la imagen completa y tú no pagas casi nada por ello.

## Instalación

```bash
npm install entroly-wasm && npx entroly-wasm
# o
pip install entroly && entroly go
```

Eso es todo. Detecta tu IDE, se conecta a Claude/Cursor/Copilot/Codex y comienza a comprimir.
