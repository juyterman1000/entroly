# Entroly

Entroly es una capa local y auditable de contexto para agentes de IA. Selecciona evidencia del repositorio con un presupuesto explícito y genera recibos que muestran qué se incluyó y qué se omitió.

```bash
pip install entroly
cd /ruta/al/repositorio
entroly verify-claims
```

`verify-claims` es una prueba local limitada de instalación y selección; no garantiza ahorro, calidad de respuesta ni facturación. Los resultados dependen de la carga de trabajo.
