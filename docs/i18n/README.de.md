# Entroly

Entroly ist eine lokale, auditierbare Kontextschicht für KI-Agenten. Sie wählt Repository-Evidenz unter einem expliziten Token-Budget aus und erstellt Belege für einbezogene und ausgelassene Inhalte.

```bash
pip install entroly
cd /pfad/zum/repository
entroly verify-claims
```

`verify-claims` ist ein begrenzter lokaler Installations- und Auswahltest. Er garantiert weder Einsparungen noch Antwortqualität oder Abrechnungsergebnisse. Resultate hängen von der Arbeitslast ab.
