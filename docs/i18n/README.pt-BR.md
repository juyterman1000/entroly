## O que você realmente ganha

| | Sem Entroly | **Com Entroly** |
|---|---|---|
| Arquivos que a IA vê | 5–10 | **Seu repositório inteiro** |
| Tokens por requisição | ~186,000 | **9,300 – 55,000** |
| Custo por 1K requisições | ~$560 | **$28 – $168** |
| Janela de contexto efetiva| 200K | **~2M (compressão de resolução variável)** |
| Custo de aprendizado | Cresce (tokens) | **$0 — provadamente token-negativo** |
| Setup | Horas de hacks | **30 segundos** |

Arquivos críticos vão na íntegra. Arquivos de suporte como assinaturas. O resto como referências. Sua IA recebe a imagem completa, e você não paga quase nada por isso.

## Instalação

```bash
npm install entroly-wasm && npx entroly-wasm
# ou
pip install entroly && entroly go
```

É só isso. Ele detecta sua IDE, se conecta ao Claude/Cursor/Copilot/Codex e começa a comprimir.
