# Entroly

Entroly é uma camada local e auditável de contexto para agentes de IA. Seleciona evidências do repositório sob um orçamento explícito de tokens e gera recibos do que foi incluído e omitido.

```bash
pip install entroly
cd /caminho/do/repositorio
entroly verify-claims
```

`verify-claims` é um teste local limitado de instalação e seleção. Não garante economia, qualidade de resposta ou resultado de cobrança; os resultados dependem da carga de trabalho.
