## Ce que vous obtenez réellement

| | Sans Entroly | **Avec Entroly** |
|---|---|---|
| Fichiers vus par l'IA | 5–10 | **Votre dépôt entier** |
| Tokens par requête | ~186,000 | **9,300 – 55,000** |
| Coût pour 1K requêtes | ~$560 | **$28 – $168** |
| Fenêtre de contexte efficace| 200K | **~2M (compression à résolution variable)** |
| Coût d'apprentissage | Augmente (tokens)| **$0 — prouvé token-négatif** |
| Configuration | Des heures de hacks | **30 secondes** |

Les fichiers critiques sont envoyés dans leur intégralité. Les fichiers de support en tant que signatures. Le reste comme références. Votre IA obtient une vue d'ensemble, et vous ne payez presque rien pour cela.

## Installation

```bash
npm install entroly-wasm && npx entroly-wasm
# ou
pip install entroly && entroly go
```

C'est tout. Il détecte votre IDE, se connecte à Claude/Cursor/Copilot/Codex et commence à compresser.
