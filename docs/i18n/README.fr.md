# Entroly

Entroly est une couche de contexte locale et auditable pour les agents IA. Elle sélectionne les preuves du dépôt sous un budget de jetons explicite et produit des reçus indiquant les éléments inclus et omis.

```bash
pip install entroly
cd /chemin/du/depot
entroly verify-claims
```

`verify-claims` est un test local limité de l’installation et de la sélection. Il ne garantit ni économies, ni qualité de réponse, ni résultat de facturation ; les résultats dépendent de la charge de travail.
