# Entroly

Entroly AI एजेंटों के लिए एक स्थानीय और ऑडिट योग्य संदर्भ परत है। यह स्पष्ट टोकन बजट के भीतर रिपॉज़िटरी साक्ष्य चुनता है और शामिल तथा छोड़े गए संदर्भ की रसीद बनाता है।

```bash
pip install entroly
cd /path/to/repository
entroly verify-claims
```

`verify-claims` केवल सीमित स्थानीय इंस्टॉल और चयन स्मोक टेस्ट है। यह बचत, उत्तर की गुणवत्ता या बिलिंग परिणाम की गारंटी नहीं देता; परिणाम वर्कलोड पर निर्भर करते हैं।
