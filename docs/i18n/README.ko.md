# Entroly

Entroly는 AI 에이전트를 위한 로컬·감사 가능 컨텍스트 계층입니다. 명시적인 토큰 예산 안에서 저장소 근거를 선택하고 포함·제외된 항목을 보여 주는 영수증을 생성합니다.

```bash
pip install entroly
cd /path/to/repository
entroly verify-claims
```

`verify-claims`는 제한된 로컬 설치 및 선택 스모크 테스트입니다. 비용 절감, 답변 품질 또는 청구 결과를 보장하지 않으며 결과는 워크로드에 따라 달라집니다.
