# Entroly

Entroly — локальный и аудируемый слой контекста для ИИ-агентов. Он выбирает доказательства из репозитория в рамках явного токен-бюджета и формирует квитанции о включённых и исключённых данных.

```bash
pip install entroly
cd /path/to/repository
entroly verify-claims
```

`verify-claims` — ограниченная локальная проверка установки и выбора контекста. Она не гарантирует экономию, качество ответа или результат расчёта стоимости. Итоги зависят от нагрузки.
